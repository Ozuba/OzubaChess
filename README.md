# OzubaChess
Ozuba Chess is a simple Chess Gui written in c++ using the SDL2 Graphics library,
it currently supports the basic part of the Chess Ruleset:

 * Only Legal Moves
 * Turn Based Play
 * Ilegal King-Exposing Moves
